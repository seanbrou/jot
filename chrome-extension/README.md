# Oat Chrome Extension

This extension opens the Oat web app in compact capture mode.

## Load it in Chrome

1. Open `chrome://extensions`
2. Turn on **Developer mode**
3. Click **Load unpacked**
4. Select this folder: `chrome-extension`

## Configure it

Open the extension's **Options** page and set the Oat web app URL:

- Local dev: `http://localhost:5173`
- Deployed app: your hosted Oat URL

## Use it

- Click the extension icon, or
- Use the Chrome command shortcut for `Open Oat quick capture`

Chrome may require you to confirm or rebind `Alt+N` in `chrome://extensions/shortcuts`.
